  • 👝 Wallet/User addresses instead of names
  • 🖍 Writes must be signed (require private key)
  • 🚫 Invalid writes will get rejected by other nodes

Nodes

Mining (Proof of Work)
  • Node connected to Blockchain Network, with access to current blockchain
  • Verify no double spending has taken place
  • Access to mempool (current unverified transactions)
  • Compute SHA(SHA(merkle root + Nonce)) to get a leading-0 result
  • Transmit results to other nodes, see if block gets accepted
  • Bad for the Environment
